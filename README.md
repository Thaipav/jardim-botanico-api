### 🌿Sistema de Gestão de Jardim Botânico

Este projeto é uma ferramenta de gestão para jardins botânicos, permitindo o controle de espécies, monitoramento de plantas individuais, registro de cuidados e integração com sensores ambientais. Desenvolvido como parte da disciplina de Programação e Arquitetura de Software.

📂 Estrutura de Arquivos

- SQL.py: Configuração da conexão com o banco de dados e criação do motor (engine).
- tabelas.py: Definição das classes usando Orientação a Objetos.
- BancoDeDados.py: Script responsável por gerar fisicamente o arquivo do banco de dados.
